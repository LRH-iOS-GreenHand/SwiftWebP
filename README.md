# SwiftWebP
WebP image decoder in Swift

## Features
- Very light
- Used in [Vulcan](https://github.com/jinSasaki/Vulcan)

## Installtion
### Carthage
Cartfile

```
github "jinSasaki/SwiftWebP"
```

## Usage

```swift
import SwiftWebP

let data // WebPData
let image = UIImage(webp: data)
```

## Requirements
- Xcode 8.1+
- Swift 3.0.1+
